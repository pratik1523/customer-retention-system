# Customer Segmentation & Retention System

## Problem Statement
Customer churn is a major challenge in the telecom industry, leading to revenue loss and increased acquisition costs. This project aims to build a production-ready analytics system to identify customer segments, predict churn risk, and recommend targeted retention strategies.

## Objectives
- Segment customers based on behavioral and service usage patterns
- Predict churn probability using machine learning
- Recommend retention actions based on customer risk and value
- Deploy the solution using APIs and interactive dashboards

## Dataset
- Telecom Customer Churn Dataset (Kaggle)

## Key Deliverables
- Customer segmentation engine
- Churn prediction model
- Retention strategy recommendation system
- FastAPI-based ML service
- Streamlit dashboard for business users
